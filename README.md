# Tetris-Terminal: A Tetris Game on terminal

<img src ="./images/tetris_play.gif">

## Overview
- A Tetris game on terminal
- Written in C++
- Requires ncurses

## Features
- Supoprts Tetris Guidelines as much as possible
- z, x -> rotate
- left, right -> shift
- down -> soft drop
- space -> hard drop
- p -> pause
- q -> quit
- r -> reset
- Combo Bonus
- T-Spin Bonus
- Back to Back Bonus
- Perfect Clear Bonus
- Up to 6 preview count
- Hold piece
- Ghost piece

## Build
- `$ make`
    - Builds nes
- `$ make test`
    - Builds nes and runs test

## Play
- `$ ./tetris`

## Platforms
- MacOS with clang
